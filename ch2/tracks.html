JCCC's Web Development and Digital Media Program has Four Different Tracks to Choose From.

Web Development and Digital Media, AAS
In just four semesters, you'll take various Web Development and Digital Media courses.
Upon completing 63 credit hours, you will graduate with an Associate of Applied Science degree
in Web Development and Digital Media.
Digital Media Certificate
Learn web scripting, HTML, web analytics and SEO through JCCC's Digital Media
certificate. You will also gain experience in visual storytelling and digital media assets.
Web Development Certificate
Gain valuable technical interface skills, server scripting, CSS techniques and more through our
Web Development Certificate.
Web Technologies Certificate
By completing 16 credits in six courses, you'll earn the Web Technologies certificate and learn
valuable skills in HTML, web scripting, digital workflow and digital media concepts.
